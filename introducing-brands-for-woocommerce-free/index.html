Installation

  1. Download the plugin from the GitHub repository.
  2. Upload the plugin ZIP file via your WordPress admin or install it manually in /wp-content/plugins/.
  3. Activate the plugin from your WordPress dashboard.

Usage

  1. Manage Brands:
    • Navigate to Products > Brands in your WordPress admin.
    • Add or edit brands with custom images and website URLs.
  2. Shortcodes:
    • Use the shortcodes to display brands or brand-related products anywhere on your site:
      • [product_brand]: Display all brands in a grid layout.
      • [brand_products brand="example-brand"]: Showcase products for a specific brand.
      • [brands_list]: Show an A-Z index of brands for easy navigation.
  3. Widgets:
    • Add the Brand Thumbnails Widget to any sidebar or footer via the WordPress Widgets screen.
  4. Custom Styles:
